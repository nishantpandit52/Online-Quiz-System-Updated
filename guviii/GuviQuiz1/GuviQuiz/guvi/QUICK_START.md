## ğŸ® How to Use in Your Code

### Basic Usage

```java
// Initialize
QuestionBank questionBank = new QuestionBank();

// Get questions (AI + Database mix)
List<Question> questions = questionBank.getQuestionsForDomain(
    "Machine Learning",  // Any domain!
    "Medium",            // Easy/Medium/Hard
    10                   // Number of questions
);

// Questions are ready to use!
for (Question q : questions) {
    System.out.println(q.getText());
    System.out.println(q.getOptions());
}
```

### Advanced: Custom Domains

```java
// Generate questions for ANY topic
questionBank.generateAndCacheAiQuestions(
    "Blockchain Technology",  // Custom domain
    "Hard",                    // Difficulty
    15                         // Count
);
```

### Disable AI Temporarily

```java
// Get only database questions
List<Question> dbQuestions = questionBank.getQuestionsForDomain(
    "Java",
    "Easy", 
    10,
    false  // Don't use AI
);
```

## ğŸ”§ Configuration File

Located at: `config.properties`

```properties

# Enable/disable AI (true/false)

# Save AI questions for reuse (true/false)

```
