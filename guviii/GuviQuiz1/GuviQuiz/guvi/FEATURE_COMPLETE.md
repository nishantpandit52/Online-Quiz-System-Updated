## âš™ï¸ Configuration

Edit `config.properties`:

```properties
# Your API key (get from makersuite.google.com)
gemini.api.key=YOUR_KEY_HERE

# Enable AI
use.ai.questions=true

# Control mix (0=no AI, 100=all AI)
ai.questions.percentage=50

# Cache for speed
cache.ai.questions=true
```
